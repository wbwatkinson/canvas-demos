Execute arbitrary js code

Problem

Canvas does not allow the use of <script> tags in the pages HTML editor. This restricts the ability to create interactive learning experiences within Canvas pages.

Solution

Though Canvas does not allow <script> tags, it does allow iFrames. The solution is to load the code in an HTML file and then load that file as an iFrame in the Canvas page. This demo will show how to use an HTML file saved in the Canvas files repository; however, it is not necessary to do so. See the following examples to load files from github or an external site:

  • Load files from github
  • Load files from an external site
  • Load files from Canvas file server. Note that this example is simply a more complex example of this one.

The workaround is to:

  1. Draft a javascript script
  2. Embed the javascript script into an HTML file
  3. Save that HTML file to the Canvas file repository
  4. Load the Canvas file as an iFrame

Caveats

This solution has the following dependency:

  1. That Canvas does not disable the ability to run arbitrary code loaded through an iFrame
