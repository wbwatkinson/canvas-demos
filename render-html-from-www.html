Render HTML from files

Problem

Canvas does not allow the use of <script> tags in the pages HTML editor. This restricts the ability to create interactive learning experiences within Canvas pages.

Solution

The workaround is to:

  1. Load a HTML file stored on the www as an iFrame

Caveats

This solution has the following dependency:

  1. That Canvas does not disable the ability to run arbitrary code loaded through an iFrame
  2. That Canvas does not prevent the use of iFrames loaded from external sources
