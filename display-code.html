Display Code with Syntax Highlighting

Problem: The Canvas page editor provides <pre> tags, but content within those tags is not rendered with syntax highlighting. However, gist.github provides a javascript a script which can be embedded in a page. This is not a complete solution, however, as Canvas does not allow using <script> in the Canvas HTML editor.

Solution: The workaround is to:

  1. Inject the appropriate <script> tags in an HTML file
  2. Save that HTML file in the Canvas file section
  3. Load that HTML file into the Canvas page with an iFrame

Note: Accessing the files saved in the Canvas file repository is non-obvious. The following is the general structure:

[course_root_url]/file_contents/course%20files/[file_path_in_canvas_files]

For example, assuming you've saved a file at test/my_example.html in a course with a url of https://lms.au.af.edu/courses/99999999/, the url to access the file is:

https://lms.au.af.edu/courses/99999999/file_contents/course%20files/test/my_example.html
