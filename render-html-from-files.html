Render HTML from files

Problem

Canvas does not allow the use of <script> tags in the pages HTML editor. This restricts the ability to create interactive learning experiences within Canvas pages.

Solution

The workaround is to:

  1. Draft a javascript script
  2. Embed the javascript script into an HTML file
  3. Save that HTML file to the Canvas file repository
  4. Load the Canvas file as an iFrame

Caveats

This solution has the following dependency:

  1. That Canvas does not disable the ability to run arbitrary code loaded through an iFrame

To function correctly in your own Canvas page, you will must:

  1. Download embed-gist-demo.html from github.
  2. Modify the course number in the downloaded file and save
  3. Create a test subfolder in your Canvas files section
  4. Save the downloaded file, modified with your course number, to your test subfolder
